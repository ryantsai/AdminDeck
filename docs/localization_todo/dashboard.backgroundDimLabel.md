# dashboard.backgroundDimLabel

- **English value**: `Dim / Lighten`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: Label for the slider in the BackgroundPopover's "Image" mode that darkens or lightens the background image. The slider ranges from −100 (darkest) through 0 (unchanged) to +100 (lightest).
- **Tone**: concise/neutral, control label
- **Placeholders**: none
- **Domain notes**: The slash separates two opposing directions of one control — negative values dim, positive values lighten. Translate both halves; keep them as a paired label.
