# dashboard.backgroundModeImage

- **English value**: `Image`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: Label for the third of three background-mode choices in the BackgroundPopover. Selecting it reveals controls to pick an image file and adjust its fit and dim/lighten.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: This mode stores an image background referenced by filename; the file is copied into a `backgrounds/` folder next to the executable.
