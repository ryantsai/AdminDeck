# dashboard.backgroundModeDefault

- **English value**: `Theme Default`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: Label for the first of three background-mode choices in the BackgroundPopover. Selecting it clears the view's custom background so the canvas uses the active color scheme background.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: "Theme" here means the app's active color scheme. This mode stores `NULL` for the view's background.
