# dashboard.backgroundFit.fill

- **English value**: `Fill`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: A `<select>` option label under the "Fit" control in the BackgroundPopover's "Image" mode. "Fill" scales the image to cover the whole canvas, cropping overflow as needed (preserves aspect ratio).
- **Tone**: concise/neutral, single-word option
- **Placeholders**: none
- **Domain notes**: Equivalent to CSS `background-size: cover`. Distinct from "Fit", which scales without cropping.
