# dashboard.backgroundFit.stretch

- **English value**: `Stretch`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: A `<select>` option label under the "Fit" control in the BackgroundPopover's "Image" mode. "Stretch" resizes the image to exactly fill the canvas, distorting its aspect ratio if needed.
- **Tone**: concise/neutral, single-word option
- **Placeholders**: none
- **Domain notes**: Equivalent to CSS `background-size: 100% 100%`. Unlike "Fill"/"Fit" it does not preserve aspect ratio.
