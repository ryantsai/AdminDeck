# dashboard.backgroundRemoveImage

- **English value**: `Remove`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `button`
- **User flow**: Button in the BackgroundPopover's "Image" mode, shown once an image is selected; clears the chosen image so the user can pick another or leave the image slot empty.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: Removes only the currently selected image reference within Image mode; it does not switch the view back to Theme Default.
