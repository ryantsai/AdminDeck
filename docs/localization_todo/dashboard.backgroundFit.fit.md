# dashboard.backgroundFit.fit

- **English value**: `Fit`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: A `<select>` option label under the "Fit" control in the BackgroundPopover's "Image" mode. "Fit" scales the image so it is fully visible within the canvas, possibly leaving empty margins (preserves aspect ratio).
- **Tone**: concise/neutral, single-word option
- **Placeholders**: none
- **Domain notes**: Equivalent to CSS `background-size: contain`. Distinct from "Fill", which crops to cover. May translate the same as the `backgroundFitLabel` ("Fit") word, but here it is one option among five.
