# dashboard.backgroundFitLabel

- **English value**: `Fit`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: Label for the `<select>` in the BackgroundPopover's "Image" mode that chooses how the background image is sized within the canvas (fill/fit/stretch/tile/center).
- **Tone**: concise/neutral, single-word control label
- **Placeholders**: none
- **Domain notes**: "Fit" here is the sizing/placement mode of an image, analogous to CSS `background-size`/`object-fit`.
