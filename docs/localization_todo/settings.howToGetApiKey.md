# settings.howToGetApiKey

- **English value**: `How to get`
- **Namespace**: `settings`
- **File/component**: `src/settings/AiSettings.tsx`
- **UI role**: `button`
- **User flow**: Appears as a small inline link next to the API key input label in AI Assistant Settings. Clicking it opens the provider's API key creation page in the user's browser.
- **Tone**: concise/neutral, action link
- **Placeholders**: none
- **Domain notes**: "API key" is a technical term that typically stays English. The phrase should be short and convey "how do I create/get one". Equivalent to "How to obtain" or "Get one here" in other UIs.
