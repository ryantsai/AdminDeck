# dashboard.backgroundFit.tile

- **English value**: `Tile`
- **Namespace**: `app`
- **File/component**: `src/dashboard/edit/BackgroundPopover.tsx`
- **UI role**: `label`
- **User flow**: A `<select>` option label under the "Fit" control in the BackgroundPopover's "Image" mode. "Tile" repeats the image at its natural size across the canvas in a grid.
- **Tone**: concise/neutral, single-word option
- **Placeholders**: none
- **Domain notes**: Equivalent to CSS `background-repeat: repeat`. Best suited to seamless/pattern images.
