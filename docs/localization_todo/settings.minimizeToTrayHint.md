# settings.minimizeToTrayHint

- **English value**: `Keeps KKTerm in the system tray when the window is minimized or closed, instead of quitting.`
- **Namespace**: `settings`
- **File/component**: `src/settings/GeneralSettings.tsx`
- **UI role**: `tooltip`
- **User flow**: Helper text under the "Minimize/Close to tray" toggle in Settings → General, explaining that minimizing or closing the window hides it to the tray rather than quitting the app.
- **Tone**: concise/neutral, explanatory
- **Placeholders**: none
- **Domain notes**: "system tray" is the OS notification area. Emphasize that the title-bar close button no longer quits when this is on; quitting is still available from the tray menu's Exit item.
