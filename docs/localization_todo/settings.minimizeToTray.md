# settings.minimizeToTray

- **English value**: `Minimize/Close to tray`
- **Namespace**: `settings`
- **File/component**: `src/settings/GeneralSettings.tsx`
- **UI role**: `label`
- **User flow**: Toggle label in Settings → General. When enabled, both minimizing and clicking the title-bar close button keep KKTerm running in the system tray instead of quitting.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: "tray" means the OS notification-area / system tray. The "/" joins the two triggers (minimize and close); keep both meanings in translation.
